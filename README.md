Workplace Principles Cards

A clean, responsive UI component designed to showcase core organizational values and workplace principles. This project uses modern CSS techniques to create an interactive, accessible, and brand-consistent experience.

✨ Features

Fully Responsive: Adapts seamlessly from mobile vertical stacks to desktop horizontal rows using Tailwind's grid/flex utilities.

Interactive States: Subtle hover effects including depth transitions (box-shadow) to enhance user engagement.

Color-Coded Branding: Uses a coordinated color palette for "Modern Awards," "Fair Work," and "Inclusion" sections.

Modern UI Patterns: Employs rounded corners, soft backgrounds, and high-contrast typography for readability.

🛠️ Tech Stack

HTML5: Semantic structure for workplace information.

Tailwind CSS: A utility-first CSS framework for rapid styling and responsive design.

Inter Font: A modern sans-serif typeface designed for screen readability.

🚀 Getting Started

Prerequisites

You don't need to install any local dependencies, as the project pulls Tailwind CSS via a Content Delivery Network (CDN).

Usage

Clone the repository or copy the index.html file.

Ensure you have an active internet connection (to load the Tailwind CDN).

Open index.html in any modern web browser.

🎨 Customization

Changing Colors

You can modify the primary brand colors by updating the arbitrary values in the Tailwind classes. For example, to change the purple accent:

Update border-[#C3ACE8] for the border color.

Update text-[#532A8C] for the heading color.

Layout Adjustment

The layout switches to a row at the md breakpoint (768px). To change this to a larger screen size, update the container class from md:flex-row to lg:flex-row.
